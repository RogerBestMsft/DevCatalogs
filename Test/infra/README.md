# Infrastructure as Code

This directory contains Bicep templates and deployment scripts for provisioning Azure resources.

## Structure

- **main.bicep** - Main template orchestrating all resources
- **modules/** - Reusable Bicep modules for individual resource types
- **configs/** - Environment-specific configuration files
- **scripts/** - Deployment and configuration scripts

## Deployment

Use the deployment scripts or follow the deployment documentation in `/docs/infra.md`.
